linux克隆虚拟机以及配置各台虚拟机，包括修改id，ssh免密登录，使用别名登录等

Posted on 2020-09-16 | In linux | | 次阅读
本文配置为hadoop搭建集群做准备，接上篇 linux配置java环境变量https://highfish920.github.io/2020/09/16/linux3/

1.克隆虚拟机

1.1 选住要克隆的虚拟机 右键》管理》克隆

因为虚拟机是开着的所以无法克隆，所以先关闭虚拟机

1.2关闭后再次选择克隆即可


选择完整克隆

设置名字和路劲

点击完成


在克隆一台


接下来启动三台虚拟机

2.配置克隆的虚拟机

2.1 开启hadoop2


2.2 输入 vi /etc/udev/rules.d/70-persistent-net.rules


修改：删除第一个，第二个把eth1改为eth0

2.3 修改ip地址
输入vi /etc/sysconfig/network-scripts/ifcfg-eth0



删除HWADDR，修改ip


2.4 同理将hadoop3修改，这里修改的ip为192.168.50.148

修改完输入 reboot 重启生效

测试ping百度也都成功

3.关闭防火墙

3.1 使用MobaXterm 登录三台虚拟机

点击这个可以同时操作

3.2 查看状态 service iptables status

3.3 关闭防火墙service iptables stop

命令：

3.4 彻底关闭防火墙以及查看
chkconfig iptables off
chkconfig --list iptables

4.关闭selinux(同样是三台一起操作)

vi /etc/selinux/config

重启三台机器 reboot

5.修改主机名

输入 vi /etc/sysconfig/network将名字改为gaoyu1

同理将hadoop机器改为gaoyu2
重启生效

6.配置别名，用别名访问

输入vi /etc/hosts配置别名

测试通过可以通过名字ping通

7.使用ssh免密登录

三台同时操作生成公钥和私钥
ssh-keygen -t rsa
按三次回车即可

接下来将第一台机器的秘钥拷贝到这三台机器，下面对第一台机器操作
输入ssh-copy-id gaoyu将秘钥拷贝到第一台机器上

接下来将其拷贝到其他两台机器

测试：
其他两台机器同理
