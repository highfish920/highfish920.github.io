赋值 vs 浅拷贝 vs 深拷贝

对象赋值
赋值是日常编程过程中最常见的操作，最简单的比如：
Student codeSheep = new Student();
Student codePig = codeSheep;
严格来说，这种不能算是对象拷贝，因为拷贝的仅仅只是引用关系，并没有生成新的实际对象：

浅拷贝属于对象克隆方式的一种，重要的特性体现在这个 「浅」 字上。
